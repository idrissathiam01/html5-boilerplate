My HTML5 Boilerplate

Why I created this?

I am attempting to re-learn web development more througly--from the bottom up by attempting Joe Santos' (aka CodingPhase.com) Coding365Challenge. I realized that I did not have a standardized method of contructing my HTML5 document for simple, one-page websites.

While I often use Bootstrap and I am familiar with h5BP, I find that those two resources might be a little bit of overkill for simple, quick projects that do not involve a lot of JavaScript or server-side scripting. I created this boilerplate to use instead of the aforementioned options.

For me, the essential goal was to have three files in the repository--index.html, main.css and global.js

What this boilerplate includes:

  • Link to Normalize.css stylesheet (@Necolas' repo on GitHub)
  • Link to custom css stylesheet: (/css/main.css)
  • Link to HTML5 Shiv--including print (@Necolas' repo on GitHub)
  • Modernizr feature detection library
  • jQuery CDN - Core 3.4.1
  • Link to file for custom global JS (/js/global.js)
